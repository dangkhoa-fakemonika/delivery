HOW TO RUN THE PROGRAM
======================
To run the program, follow 1 of the following 2 methods:
1. Simply double-click on the "run.bat" file.
2. Open cmd in THIS directory and type "python main.py".

After that, give the input file name and option to use the customized textures.
If the program is not running, please head to the "requirements.txt" file and install all the required libraries.
